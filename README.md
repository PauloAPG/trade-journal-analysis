# 📊 Trade Journal Analysis

Análise de performance de operações no mercado financeiro (XAU/USD e BTC/USD)
utilizando Python, Pandas e Matplotlib.

## 🎯 Objetivo
Identificar padrões de performance com base em dados reais de operações,
respondendo perguntas como:
- Em qual sessão (Londres, Nova York, Ásia) tenho melhor desempenho?
- Qual gatilho (FVG, IFVG, BPR) tem maior taxa de acerto?
- Como o estado emocional impacta o resultado?
- Qual é a minha curva de capital ao longo do tempo?

## 🛠️ Tecnologias
- Python 3
- Pandas
- Matplotlib
- Jupyter Notebook
